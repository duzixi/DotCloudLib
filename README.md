DotCloudLib: A Point Cloud Post-Processing Algorithm Library

点云后处理算法库DotCloudLib是基于C#实现的点云后处理算法类库。主要用于在点云标准化预处理的基础上，进行三维空间分析和处理。本库所有算法自主原生开发，仅引用自研算法库，不依赖第三方技术，可直接在.NET工程、Unity3D工程或原生C#程序使用。算法库名“Dot”援引.NET平台的“.”，中文双关“点”，意为点云库。目前，公开的版本为1.0版，算法库源码参见src文件夹。需要引用的dll参见dll文件夹。后续算法库开源计划，以及算法库具体功能内容参见 doc 文件夹中的说明文档。

DotCloudLib is a C#-implemented point cloud post-processing algorithm library. It is mainly used for 3D spatial analysis and processing based on standardized point cloud preprocessing.All algorithms in this library are independently and originally developed. It only references self-developed algorithm components without relying on any third-party technologies, and can be directly applied in .NET projects, Unity3D projects and native C# programs.The prefix Dot in the library name derives from the dot symbol of the .NET platform, and also corresponds to the Chinese character meaning "point", highlighting its positioning as a professional point cloud library.The current public release is Version 1.0. The source code of the algorithm library is available in the src folder, and the required referenced DLL files are placed in the dll folder. For the subsequent open-source roadmap and detailed functional specifications of the library, please refer to the documents in the doc folder.
